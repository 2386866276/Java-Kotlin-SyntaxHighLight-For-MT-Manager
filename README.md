# Java-Kotlin-SyntaxHighLight-For-MT-Manager
Java-Kotlin-SyntaxHighLight-For-MT-Manager 专为 MT 管理器开发的 Java 与 Kotlin 语法高亮方案，提供定制化浅色/深色双模式配色。覆盖字符串、数字、注释（高亮 TODO/FIXME 等标记）、注解、运算符、控制流关键字、类型名、函数名及成员访问等全部语法单元，支持 .java/.jsp/.kt/.kts 文件。规则基于 MT 内置引擎精准匹配 Java 文本块与 Kotlin 多行字符串，用户可自行修改配色。适用替换默认高亮、获取 IDE 风格专属阅读体验。项目以 Apache-2.0 许可开源，由原作者林映雪设计、2386866276 修改维护。
